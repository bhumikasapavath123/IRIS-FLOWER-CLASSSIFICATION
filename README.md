# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

The **Iris Flower Classification** project is a beginner-friendly machine learning application that classifies iris flowers into one of three species based on their physical measurements. It uses the famous **Iris Dataset** and a **Decision Tree Classifier** to predict the flower species with high accuracy.

This project demonstrates the complete machine learning workflow, including data loading, preprocessing, model training, evaluation, and prediction.

---

# 🎯 Objectives

* Classify iris flowers into different species.
* Learn the fundamentals of supervised machine learning.
* Understand classification algorithms.
* Build a simple yet effective prediction model.
* Gain hands-on experience with the Scikit-learn library.

---

# 🌼 Flower Species

The model predicts one of the following species:

* 🌸 Iris Setosa
* 🌺 Iris Versicolor
* 🌷 Iris Virginica

---

# ✨ Features

* Uses the built-in Iris dataset from Scikit-learn.
* Implements the Decision Tree Classification algorithm.
* Predicts flower species based on four input features.
* Displays model accuracy.
* Generates a detailed classification report.
* Allows users to enter flower measurements for real-time predictions.
* Easy to understand and beginner-friendly.

---

# 📊 Dataset Information

The Iris dataset contains **150 flower samples** with **4 features**.

| Feature      | Description              |
| ------------ | ------------------------ |
| Sepal Length | Length of the sepal (cm) |
| Sepal Width  | Width of the sepal (cm)  |
| Petal Length | Length of the petal (cm) |
| Petal Width  | Width of the petal (cm)  |

**Target Variable**

* Iris Setosa
* Iris Versicolor
* Iris Virginica
